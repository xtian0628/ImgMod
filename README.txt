1. 'NN_model.m': the neural network model structure 

2. 'fit_NN_model.m': fitting the neural network model using previous MEG data (Tian & Poeppel, 2013).

3. 'neural_output_before_IMG_adap.m': deriving neural output for fitting baseline responses in the behavioral experiemnt 

4. 'beh_model_from_NN_output.m': procedure of fitting baseline responses to /ba/-/da/ continumm using the peak latency of the simulated neural responses

5. 'fit_beh_model.m': perform fitting using baseline responses to /ba/-/da/ continumm

6. 'neural_output_after_IMG_adap.m': deriving neural output for generating prediction of the behavioral experiemnt (modulation effects of imagery to the perception of /ba/-/da/ continum)

7. 'generate_predictions.m': generating model predictions of modulation effects of imagery to the perception of /ba/-/da/ continum for the behavioral experiment

* .m files are scripts for modeling; 
* .mat files are generated data from .m files for subsequent simulations
